⚡ AI is Secured via Server
Your Groq API key is stored in Vercel environment variables — it is never sent to the browser. Add it once in Vercel settings and AI works for everyone securely.
1
Go to console.groq.com → sign up free, no card needed
2
Click API Keys → Create API Key → copy it (starts with gsk_)
3
Go to your Vercel Dashboard → your project → Settings → Environment Variables
4
Add: Name = GROQ_API_KEY · Value = your key · Click Save → Redeploy
